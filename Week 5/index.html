About API's

What are API's?

The word API is an abreviation for Application Programming Interface. Its main purpose is to either send or receive data from another server to your application or use.

What are the different versions of API's?

There are two different versions: RESTful and Soap. RESTful is the more modern approach. It stands for Representational State Transfer. It relies on client-server protocall and treats server objects as objects that can be created or destroyed. RESTful API's have the ability to GET, PUT, POST, and DELETE information. Soap on the otherhand is still used by developers today, but is less common compared to RESTful. Soap retrives a XML document that contains the envelope, header, body, and fault. Soap can insert, update, delete, or select information from a database.
